# Проекты для реальных заказчиков

Отличительная особенность этих проектов — стремление создать решение, которое заказчик смог бы применить с пользой.  
Не всегда заказчики явно озвучивают или акцентируют внимание на некоторых проблемах.  
Выявление и решение подобных задач есть в публикуемых здесь проектах.


| Проект                | Описание              | Решение                   |
|:----------------------|:----------------------|:--------------------------|
| [English Movie Level Prediction: Research and App](https://github.com/Nanobelka/english_subtitles_level) | Приложение на платформе Streamlit, основанное на модели машинного обучения, для предсказания CEFR-уровня английского языка по субтитрам к фильму/сериалу – для всех, кто изучает английский язык | [веб-приложение](https://movie-level.streamlit.app/) |
